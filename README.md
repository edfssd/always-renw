# always-renw

TG_BOT_TOKEN：你的 Telegram 机器人的 Token（例如：123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ）。

TG_CHAT_ID：你的 Telegram 个人账户 ID 或接收通知的频道/群组 ID（例如：987654321）。

进入你的 GitHub 仓库 -> 点击顶部的 Settings -> 左侧菜单展开 Secrets and variables -> 点击 Actions。

点击右侧的 New repository secret 按钮，添加以下三个密钥：

ALWAYSDATA_USER：填入你的 Alwaysdata 登录邮箱（例如 ssddd@.eu.org）。

ALWAYSDATA_PASS：填入对应的登录密码（例如sdsa）。

REPO_TOKEN：填入你的 GitHub Personal Access Token (PAT)。
